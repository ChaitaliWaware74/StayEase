# StayEase – Smart Accommodation Finder Platform

StayEase is a full-stack web application built using the MERN stack that helps users find and post rental accommodations such as flats, PGs, and shared rooms.

---

## 🚀 Features

- User Registration & Login (JWT Authentication)
- Add, Edit, Delete Listings
- Upload Multiple Images
- Search & Filter Listings (Location, Rent, Gender)
- View Detailed Property Information
- Amenities (Parking, Water, Electricity)
- Google Maps Integration (Location View)
- User Dashboard (My Listings)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Tools & Libraries
- Axios
- JWT (Authentication)
- bcrypt.js

## ⚙️ Installation & Setup

### 1. Clone the repository


git clone https://github.com/ChaitaliWaware74/StayEase.git


---

### 2. Install dependencies

#### For backend:

cd server
npm install


#### For frontend:

cd client
npm install


---

### 3. Run the project

#### Start backend:

cd server
node server.js


#### Start frontend:

cd client
npm run dev
